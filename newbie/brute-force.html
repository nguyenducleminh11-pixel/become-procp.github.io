Brute Force (Vét cạn) Brute Force Algorithm

Khái niệmConcept

Thuật toán Brute Force là phương pháp giải quyết vấn đề bằng cách thử tất cả các khả năng có thể cho đến khi tìm ra đáp án. Nó tốn thời gian nhưng đảm bảo luôn tìm thấy kết quả nếu tồn tại.

Brute Force is a straightforward approach to solving a problem by trying all possible solutions until the correct one is found. It is time-consuming but guarantees a result if one exists.

Ví dụ: Tìm cặp số có tổng bằng SExample: Two Sum (S)

Bài toán: Cho mảng A gồm N phần tử và số S. Tìm cặp số (i, j) sao cho A[i] + A[j] = S.
Problem: Given an array A of N elements and a value S. Find a pair (i, j) such that A[i] + A[j] = S. 
#include <iostream>
#include <vector>

void solve() {
    int n = 5, s = 10;
    int a[] = {2, 4, 6, 8, 3};

    // Brute Force: Thử mọi cặp (i, j)
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            if (a[i] + a[j] == s) {
                std::cout << a[i] << " + " << a[j] << " = " << s << std::endl;
            }
        }
    }
}
